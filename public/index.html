What makes it different

Not a template. Not a wrapper. A multi-agent pipeline that actually reads your GitHub and the JD.

FAQ

Which roles does this work for?

Every technical role at the pilot stage — Software Engineer (any stack), Data Analyst, Data Scientist, ML Engineer, DevOps, Cloud, Cybersecurity, QA. The multi-agent pipeline extracts the JD role automatically; no hard-coding. Non-technical roles (Marketing, Design, Finance) work too but aren't optimized in v1.

What language do I chat in?

Hinglish, English, or a mix. Our bot Saathi detects and responds in the same style. The final PDF is always in professional English.

Do I need a GitHub account?

Strongly recommended for technical roles — the README gives us the material to write specific, metric-driven bullets. You can decline, but you'll get thinner bullets.

What if I don't have any work experience?

That's fine — the pipeline is built for college students. Projects, positions of responsibility, hackathons, and certifications carry equal weight for a fresher resume.

How is my data handled?

Your phone number is hashed before storage. Your resume content is used only to build your PDF and is deleted on request within 30 days. Full detail in the Privacy Policy.

What's your refund policy?

Full refund if the clean PDF is never delivered within 24 hours of your ₹49 payment, or if the file has a technical fault we can't fix. Not available for subjective dissatisfaction with wording (that's what the edit loop is for). Full detail in the Terms of Service.
